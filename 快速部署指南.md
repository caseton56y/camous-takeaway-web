## 🚀 最快上线方式（5分钟搞定）

### 方式 1：使用 Vercel 部署（推荐，完全免费）

#### 步骤：

1. **注册 GitHub 账号**
   - 访问 https://github.com 注册一个免费账号

2. **在 GitHub 创建仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名：`campus-takeaway-web`
   - 选择 Public（公开）
   - 点击 "Create repository"

3. **推送代码到 GitHub**
   在项目目录中打开 PowerShell，依次运行：
   ```powershell
   cd C:\Users\qw214\Desktop\campus-takeaway-web
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/campus-takeaway-web.git
   git push -u origin main
   ```

4. **使用 Vercel 部署**
   - 访问 https://vercel.com
   - 使用 GitHub 账号登录
   - 点击 "New Project"
   - 选择您的 `campus-takeaway-web` 仓库
   - 直接点击 "Deploy"（无需任何配置）
   - 等待 1-2 分钟
   - 完成！获得一个类似 `https://campus-takeaway-web.vercel.app` 的网址

---

### 方式 2：使用 Netlify 部署（同样免费）

1. 访问 https://netlify.com
2. 注册并使用 GitHub 登录
3. 点击 "Add new site" → "Import an existing project"
4. 连接 GitHub
5. 选择您的仓库
6. 点击 "Deploy site"
7. 完成！获得一个临时网址

---

### 方式 3：直接上传构建产物

如果您已经有网站服务器：

1. 构建产物在 `C:\Users\qw214\Desktop\campus-takeaway-web\dist` 目录
2. 将 `dist` 目录中的**所有文件**上传到您的服务器
3. 配置您的 Web 服务器（Nginx/Apache 等）指向这个目录
